# סיכום תפקידים במערכת Shipment Bot

## 1. שולח (Sender) 📦

### מה זה?
משתמש שרוצה לשלוח חבילה ממקום למקום. זה הלקוח הבסיסי של המערכת.

### רישום
- שם מלא
- מספר טלפון

### אפשרויות בתפריט
- **➕ משלוח חדש** — יצירת משלוח עם כתובת איסוף, כתובת יעד, דחיפות, מחיר ותיאור
- **📦 המשלוחים שלי** — צפייה במשלוחים שיצר
- **🚚 הצטרפות למנוי וקבלת משלוחים** — מעבר לתפקיד שליח
- **🚗 הצטרפות כנהג** — מעבר לתפקיד נהג
- **🏪 הצטרפות כתחנה** — מעבר לתפקיד בעל תחנה
- **📞 פנייה לניהול** — יצירת קשר עם המערכת

### זרימת יצירת משלוח
1. עיר איסוף → רחוב → מספר בית → דירה
2. סוג משלוח (עירוני/בין-עירוני)
3. עיר יעד → רחוב → מספר בית → דירה
4. דחיפות (מיידי / מאוחר יותר) → בחירת שעה (אם לא מיידי)
5. מחיר
6. תיאור המשלוח
7. אישור סופי

---

## 2. שליח (Courier) 🚚

### מה זה?
שליח שמבצע משלוחים בפועל — אוסף חבילות ומוסר אותן. עובר תהליך KYC (זיהוי) לפני שמתחיל לעבוד.

### רישום (KYC)
1. שם מלא
2. העלאת תעודת זהות (תמונה)
3. צילום סלפי
4. בחירת סוג רכב (אופנוע, רכב, ואן וכו')
5. צילום הרכב
6. אישור תנאי שימוש
7. **המתנה לאישור אדמין** — האדמין מאשר/דוחה/חוסם

### אפשרויות בתפריט (אחרי אישור)
- **💰 מצב הארנק** — יתרה נוכחית, מסגרת אשראי
- **📍 הגדרות אזור** — שינוי אזור שירות
- **📦 היסטוריית עבודות** — משלוחים שביצע בעבר
- **📦 משלוח פעיל** — משלוח שנמצא כרגע בטיפולו
- **💳 הפקדה** — טעינת ארנק (ביט/פייבוקס/העברה בנקאית)
- **❓ תמיכה** — פנייה לתמיכה
- **🏪 תפריט סדרן** — מופיע רק אם השליח הוא גם סדרן בתחנה

### זרימת עבודה על משלוח
1. צפייה במשלוחים זמינים (בקבוצת טלגרם/וואטסאפ)
2. תפיסת משלוח (לחיצה על כפתור בקבוצה)
3. אישור תפיסה
4. סימון "אספתי" — המשלוח עובר לסטטוס "בדרך"
5. סימון "מסרתי" — המשלוח הושלם

### ארנק
- לכל שליח יש ארנק דיגיטלי
- יש מסגרת אשראי (ניתן לעבוד גם ביתרה שלילית עד גבול מסוים)
- ניתן לטעון דרך ביט/פייבוקס/העברה בנקאית
- העלאת צילום מסך של אישור ההעברה

## 3. נהג (Driver / iDriver) 🚗

### מה זה?
נהג שמחפש נסיעות (לא משלוחים!) — מערכת נפרדת לחיפוש והתאמת נסיעות בין ערים. מיועד בעיקר לקהילה החרדית (יש אימות קוד לבוש).

### רישום
1. שם מלא
2. תאריך לידה
3. תיאור רכב
4. קוד לבוש (חסיד שחור לבן / חרדי שחור לבן / חרדי מודרני / דתי אלגנט / דתי מעורב / חילוני)
5. **אימות חרדי** (לפי קוד הלבוש):
   - צילום סלפי
   - תעודת זהות
   - המתנה לאישור מנהל

### אפשרויות בתפריט
- **🛠 הגדרות חיפוש** — סוג רכב, סוג נסיעה, מסגרת זמן, מצב חיפוש עתידי בלבד
- **🔍 חיפושים פעילים** — צפייה וניהול חיפושים פעילים
- **📖 הוראות שימוש** — מדריך
- **💳 מנוי** — צפייה ורכישת מנוי

### פקודות טקסט (מהתפריט)
- **`פ <יעד>`** — חיפוש נסיעה ליעד
- **`פ <מוצא> <יעד>`** — חיפוש ממוצא ספציפי
- **`פ א <יעד>`** — חיפוש אזורי
- **`פ מיקום`** — חיפוש לפי GPS
- **`ע`** — השהיית כל החיפושים
- **`ה`** — חידוש חיפושים מושהים
- **`מ`** — מחיקת חיפוש בודד
- **`ממ`** — מחיקת כל החיפושים
- **`מחירון <מוצא> <יעד>`** — בדיקת מחירון
- **`<מוצא> <יעד> <מקומות> מק <מחיר> ש״ח`** — פרסום נסיעה

### מנוי
- יש מערכת מנויים — נהג צריך מנוי פעיל כדי לחפש נסיעות
- סשן 24 שעות — חיפושים פעילים למשך יום

## 4. סדרן (Dispatcher) 🏪

### מה זה?
מפעיל של תחנת שליחויות — מנהל משלוחים מטעם התחנה, מחייב נהגים, ומפרסם נסיעות. הסדרן הוא בעצם שליח שקיבל גם הרשאת סדרן בתחנה מסוימת.

### איך נהיים סדרן?
בעל תחנה מוסיף את מספר הטלפון של השליח כסדרן בתחנה שלו.

### אפשרויות בתפריט
- **➕ הוספת משלוח** — יצירת משלוח מטעם התחנה (עם עמלה)
- **📦 משלוחים פעילים** — צפייה במשלוחים פעילים של התחנה
- **📋 היסטוריית משלוחים** — היסטוריה
- **💳 חיוב ידני** — חיוב נהג באופן ידני (שם + סכום + תיאור)
- **🚗 פרסום נסיעה** — פרסום נסיעה (מוצא → יעד → מקומות → מחיר)
- **🛣 נסיעות פעילות** — צפייה בנסיעות שפורסמו
- **🔙 חזרה לתפריט ראשי** — חזרה לתפריט השליח הרגיל

### זרימת הוספת משלוח
1. עיר איסוף → רחוב → מספר בית
2. עיר יעד → רחוב → מספר בית → דירה
3. תיאור המשלוח
4. עמלה (fee)
5. אישור

### זרימת חיוב ידני
1. שם הנהג (בחירה מרשימה)
2. סכום
3. תיאור
4. אישור

---

## 5. בעל תחנה (Station Owner) 🏢

### מה זה?
מי שמנהל תחנת שליחויות — שולט בהגדרות, מנהל סדרנים ובעלים נוספים, רואה דוחות כספיים, מנהל רשימה שחורה של שליחים בעייתיים.

### אפשרויות בתפריט
- **👤 ניהול בעלים** — הוספה/הסרה של בעלים נוספים לתחנה (לפי מספר טלפון)
- **👥 ניהול סדרנים** — הוספה/הסרה של סדרנים (dispatchers) לתחנה
- **💰 ארנק תחנה** — צפייה ביתרה + שינוי אחוז עמלה
- **📊 דוח גבייה** — דוח כספי על גביות
- **🚫 רשימה שחורה** — חסימת/שחרור שליחים בעייתיים (טלפון + סיבה)
- **⚙️ הגדרות קבוצות** — הגדרת קבוצת טלגרם/וואטסאפ ציבורית ופרטית
- **🏪 הגדרות תחנה** — עריכת שם, תיאור, שעות פעילות, אזורי שירות

### ניהול בעלים
- הוספת בעלים חדש לפי מספר טלפון
- הסרת בעלים (בחירה מרשימה + אישור)

### ניהול סדרנים
- הוספת סדרן לפי מספר טלפון
- הסרת סדרן (בחירה מרשימה + אישור)

### רשימה שחורה
- הוספה: מספר טלפון + סיבת חסימה
- הסרה: בחירה מרשימה + אישור

### הגדרות קבוצות
- קבוצה ציבורית — בה מפורסמים המשלוחים הזמינים
- קבוצה פרטית — לתקשורת פנימית

### הגדרות תחנה
- שם התחנה
- תיאור
- שעות פעילות
- אזורי שירות

---

## 6. אדמין (Admin) 👑

### מה זה?
מנהל המערכת — מאשר/דוחה/חוסם שליחים, יכול לעבור בין תפקידים לצורך בדיקות.

### אפשרויות
- אישור/דחייה של שליחים חדשים (KYC)
- חסימת שליחים
- בחירת תפקיד (SELECT_ROLE) — לצורך בדיקות, יכול "להתחזות" לכל תפקיד

---

## קשרים בין התפקידים

```
שולח ──יוצר משלוח──▶ משלוח (סטטוס: פתוח)
                         │
                         ▼
              סדרן מאשר (אם דרך תחנה)
                         │
                         ▼
              שליח תופס את המשלוח
                         │
                         ▼
              שליח אוסף ──▶ שליח מוסר
                                │
                                ▼
                          משלוח הושלם
```

### היררכיה של תחנה
```
בעל תחנה (Station Owner)
    │
    ├── מנהל סדרנים (מוסיף/מסיר)
    ├── מנהל בעלים נוספים
    ├── מנהל רשימה שחורה
    └── רואה דוחות כספיים
         │
         ▼
    סדרן (Dispatcher)
    │
    ├── מוסיף משלוחים מטעם התחנה
    ├── מחייב נהגים ידנית
    └── מפרסם נסיעות
         │
         ▼
    שליח (Courier)
    │
    ├── תופס משלוחים מהתחנה
    ├── מבצע איסוף ומסירה
    └── מנהל ארנק
```

### מעבר בין תפקידים
- **שולח → שליח**: דרך כפתור "הצטרפות למנוי וקבלת משלוחים" + KYC
- **שולח → נהג**: דרך כפתור "הצטרפות כנהג" + רישום נהג
- **שולח → תחנה**: דרך כפתור "הצטרפות כתחנה"
- **שליח → סדרן**: בעל תחנה מוסיף את השליח כסדרן (תפקיד כפול)

---

## סטטוסים של משלוח

| סטטוס | משמעות | מי גורם |
|--------|---------|----------|
| open | פתוח, מחכה לשליח | שולח/סדרן יצר |
| pending_approval | ממתין לאישור סדרן | שיוך לתחנה |
| captured | נתפס ע"י שליח | שליח לחץ "תפוס" |
| in_progress | בדרך | שליח סימן "אספתי" |
| delivered | נמסר | שליח סימן "מסרתי" |
| cancelled | בוטל | שולח/סדרן ביטל |

## סטטוסי אישור שליח

| סטטוס | משמעות |
|--------|---------|
| pending | ממתין לאישור אדמין |
| approved | מאושר — יכול לעבוד |
| rejected | נדחה — יכול להגיש מחדש |
| blocked | חסום — לא יכול להגיש מחדש |
